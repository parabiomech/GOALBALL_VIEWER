# 📸 스크린샷 촬영 가이드

이 문서는 사용자 매뉴얼에 필요한 스크린샷을 촬영하는 방법을 안내합니다.

## 📂 저장 경로
모든 스크린샷은 다음 경로에 저장해주세요:

## 📋 필요한 스크린샷 목록

### 1. 파일 불러오기 화면
**파일명**: `01_file_upload.png`

**촬영 방법**:
1. index.html을 브라우저에서 열기
2. "📁 데이터 파일 불러오기" 섹션 캡처
3. "파일 선택" 버튼과 설명 텍스트가 모두 보이도록 촬영

**캡처 영역**:
- "📁 데이터 파일 불러오기" 제목
- 점선 테두리 박스 전체
- "파일 선택" 버튼
- 안내 문구

---

### 2. 경기 정보
**파일명**: `02_match_info.png`

**촬영 방법**:
1. 샘플 데이터 파일 불러오기
2. "📊 경기정보" 섹션 캡처
3. 경기 날짜, 경기명, 팀 이름 등이 표시된 상태로 촬영

**캡처 영역**:
- "📊 경기정보" 제목
- 경기 날짜, 경기명
- A팀/B팀 정보
- 총 기록 수

### 3. 궤적 그래프
**파일명**: `03_trajectory_map.png`

**촬영 방법**:
1. 데이터가 로드된 상태에서 궤적 그래프 섹션 캡처
2. 투구 궤적 선과 점들이 잘 보이도록 촬영
3. A팀(빨강)과 B팀(파랑) 데이터가 모두 보이는 것이 좋음

**캡처 영역**:
- "궤적 그래프" 제목
- 코트 전체 (18m × 9m)
- 골대 양쪽
- 투구 궤적 선들
- 하단의 설명 라벨

## 📸 촬영 팁

### 1. 해상도
- 최소 1280×720 해상도로 촬영
- 고해상도일수록 좋음 (1920×1080 권장)

### 2. 브라우저
- 개발자 도구는 닫은 상태로 촬영
- 확대/축소는 100% (기본값)
- 전체 화면 모드는 사용하지 않음

### 3. 캡처 도구
**Windows**:
- `Win + Shift + S`: 영역 캡처
- Snipping Tool
- ShareX (고급)

**Mac**:
- `Cmd + Shift + 4`: 영역 캡처
- `Cmd + Shift + 5`: 캡처 옵션

### 4. 이미지 포맷
- PNG 형식 권장 (손실 없음)
- JPG도 가능 (품질 90% 이상)

### 5. 파일명 규칙
- 반드시 지정된 파일명 사용
- 소문자 사용
- 언더스코어(_) 사용

### 6. 화질
- 텍스트가 선명하게 읽힐 수 있도록
- 색상이 정확하게 표현되도록
- 적절한 밝기 유지
