## 개요

**골볼 데이터 뷰어 Ver5**는 골볼 경기의 데이터를 분석하고 시각화하는 웹 기반 도구입니다.

### 주요 특징
- ✅ JSON/CSV 파일 불러오기
- ✅ 실시간 코트 시각화
- ✅ 투구 궤적 및 히트맵 분석
- ✅ 다양한 필터 옵션
- ✅ 통계 및 패턴 분석
- ✅ 데이터 내보내기 (JSON/CSV)
- ✅ 이메일 발송 기능

---

## 시작하기

### 1. 데이터 파일 불러오기

![파일 불러오기 화면](./screenshots/01_file_upload.png)

#### 사용 방법:
1. **"파일 선택"** 버튼을 클릭합니다
2. JSON 또는 CSV 파일을 선택합니다
3. 파일이 성공적으로 로드되면 경기 정보와 시각화 화면이 표시됩니다

#### 지원 파일 형식:
- **JSON**: Tool에서 내보낸 JSON 형식 파일
- **CSV**: 표준 CSV 형식 파일

## 주요 기능

### 📊 경기 정보 표시

![경기 정보](./screenshots/02_match_info.png)

파일을 불러오면 다음 정보가 표시됩니다:

## 데이터 시각화

### 1. 궤적 그래프

![궤적 그래프](./screenshots/03_trajectory_map.png)

#### 기능:
- 모든 투구의 궤적을 코트 위에 시각화
- 팀별로 색상 구분 (A팀: 빨강, B팀: 파랑)
- 이동 → 출발 → 릴리스 → 방향 순서로 표시

#### 구역 정보:
- **9개 구역**: 각 구역은 90x45 픽셀로 구성
- **Goal/Out/Play** 영역으로 구분

---

### 2. 히트맵 (8×8 영역 빈도)

![히트맵](./screenshots/04_heatmap.png)

#### 기능:
- 8×8 그리드로 코트를 분할
- 각 영역별 투구 빈도를 색상으로 표시
- 밀도가 높을수록 진한 색상으로 표시
- 빈도 숫자도 함께 표시

#### 색상 범례:
- 낮은 빈도: 투명/연한 색
- 높은 빈도: 진한 색 (팀 색상)

---

### 3. 기본 통계

![기본 통계](./screenshots/05_basic_stats.png)

#### 표시 정보:
- **총 투구 수**: 전체 기록된 투구 횟수
- **팀별 통계**: A팀 vs B팀 투구 비교 (바 차트)
- **포지션별 통계**: L(왼쪽), C(중앙), R(오른쪽) 분포
- **기술 유형 분석**: F, SB, 2B, 3B, 4B 등
- **선수별 통계**: 선수별 투구 횟수
- **이벤트 통계**: G(골), H(히트), L(로스), TP 등

---

### 4. 시간별 이벤트 그래프

![시간별 이벤트](./screenshots/06_timeline_chart.png)

#### 기능:
- X축: 경기 시간 (초)
- Y축: 투구 순서
- 각 점은 투구 기록 시점을 나타냄
- 팀별로 색상 구분

#### 활용:
- 경기 흐름 파악
- 특정 시간대 투구 빈도 분석
- 팀별 투구 타이밍 비교

---

### 5. 포지션 시퀀스 분석

![포지션 시퀀스](./screenshots/07_position_sequence.png)

#### A팀 / B팀 시퀀스 차트:
- 투구 순서에 따른 포지션 변화를 그래프로 표시
- L(왼쪽), C(중앙), R(오른쪽) 포지션 추적

#### 포지션 빈도:
- 각 포지션별 투구 횟수와 비율 표시
- 팀의 포지션 선호도 분석

---

### 6. 투구코스 패턴 TOP 15

![투구코스 패턴](./screenshots/08_top_patterns.png)

#### 기능:
- 가장 많이 사용된 투구 패턴 15개 표시
- 패턴 구성: 이동 → 출발 → 릴리스 → 방향
- 각 패턴의 사용 횟수와 비율 표시

#### 패턴 표기법:
- 예: `A-03 → A-02 → A-01 → A-01` (24회, 12.3%)
- 각 단계별 구역 코드로 표시

## 필터 및 분석

### 필터 설정 패널

![필터 패널](./screenshots/09_filter_panel.png)

#### 1. 팀 선택
- ☑️ A팀
- ☑️ B팀
- 전체선택 버튼으로 한 번에 선택/해제

#### 2. 시간 선택
- ☑️ 전반
- ☑️ 후반
- ☑️ 연장전

#### 3. 선수 선택
- 1번 ~ 19번 선수 개별 선택
- 전체선택 버튼 제공

#### 4. 포지션 선택
- ☑️ L (왼쪽)
- ☑️ C (중앙)
- ☑️ R (오른쪽)

#### 5. 기술 유형
- F, SB, 2B, 3B, 4B, TF, TSB, T2B, T3B, T4B, IC, OC, 미지정

#### 6. 이벤트
- G (골), H (히트), L (로스), TP, 미지정

#### 7. 이벤트2
- FB (파울볼), PT (패널티), 미지정

### 필터 적용
"**필터 적용**" 버튼을 클릭하면 선택한 조건에 맞는 데이터만 시각화됩니다.

## 데이터 내보내기

### 1. JSON 내보내기

![JSON 내보내기](./screenshots/11_export_json.png)

#### 사용 방법:
1. **"JSON 내보내기"** 버튼 클릭
2. 파일 저장 위치 선택
3. 파일명 형식: `goalball_data_YYYYMMDD_HHMMSS.json`

#### 포함 데이터:
- 경기 정보
- 모든 투구 기록
- 포지션 맵
- 타임스탬프

---

### 2. CSV 내보내기

![CSV 내보내기](./screenshots/12_export_csv.png)

#### 사용 방법:
1. **"CSV 내보내기"** 버튼 클릭
2. 파일 저장 위치 선택
3. 파일명 형식: `goalball_data_YYYYMMDD_HHMMSS.csv`

#### CSV 컬럼:
- ID, 시간, 팀, 선수, 포지션, 시간대, 기술, 이벤트, 이벤트2
- 패턴, 경기시간
- 이동X, 이동Y, 출발X, 출발Y, 릴리스X, 릴리스Y, 방향X, 방향Y

---

### 3. 이메일 발송

![이메일 발송](./screenshots/13_email_dialog.png)

#### 사용 방법:
1. **"이메일 발송"** 버튼 클릭
2. 이메일 다이얼로그에서 정보 입력:
   - 받는 사람 이메일
   - 보내는 사람 이메일
   - 파일 형식 선택 (JSON/CSV)
   - 메시지 (선택사항)
3. **"발송"** 버튼 클릭

#### 주의사항:
- 유효한 이메일 주소를 입력해야 합니다
- 메시지는 선택사항입니다

---

### 4. 시각화 지우기

![시각화 지우기](./screenshots/14_clear_visual.png)

## 팁과 요령

### 💡 효율적인 분석 방법

#### 1. 특정 선수 분석
- 필터에서 해당 선수만 선택
- 해당 선수의 투구 패턴과 성공률 확인

#### 2. 포지션별 전략 분석
- 포지션 필터를 활용하여 L, C, R 각각 분석
- 각 포지션에서의 성공 패턴 파악

#### 3. 시간대별 분석
- 전반/후반 필터 활용
- 경기 후반부 투구 패턴 변화 추적

#### 4. 골 패턴 연구
- 이벤트 필터에서 'G(골)'만 선택
- 골로 이어진 투구 패턴 집중 분석

#### 5. 팀 간 비교
- 한 팀씩 선택하여 개별 분석
- 패턴과 통계 비교

## 구역 코드표

### 코트 구역 (9개)

```
┌──────────────────────┐
│  A-01  │  A-02  │  A-03  │  (왼쪽 골대)
├──────────────────────┤
│  A-11  │  A-12  │  A-13  │
├──────────────────────┤
│  A-21  │  A-22  │  A-23  │  (중앙)
├──────────────────────┤
│  B-11  │  B-12  │  B-13  │
├──────────────────────┤
│  B-01  │  B-02  │  B-03  │  (오른쪽 골대)
└──────────────────────┘
```

- **A-0x**: A팀 골 라인
- **A-1x**: A팀 진영
- **A-2x**: 중앙 라인
- **B-1x**: B팀 진영
- **B-0x**: B팀 골 라인

## 문제 해결

### 파일이 로드되지 않을 때
- 파일 형식이 JSON 또는 CSV인지 확인
- 파일 내용이 올바른 구조인지 확인
- 브라우저 콘솔에서 오류 메시지 확인

### 시각화가 표시되지 않을 때
- 데이터가 제대로 로드되었는지 확인
- "필터 적용" 버튼을 클릭해보기
- 필터 설정이 너무 제한적이지 않은지 확인

### 내보내기가 작동하지 않을 때
- 브라우저의 다운로드 권한 확인
- 팝업 차단 설정 확인
- 다른 브라우저에서 시도
